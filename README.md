# Password Strength Checker

## Description
A Python-based Password Strength Checker that evaluates password strength using:
- Length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## Features
- Password scoring
- Strength classification
- Beginner-friendly cybersecurity project

## How to Run

```bash
python password_checker.py
```
